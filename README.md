# script-language-fibo-benchmarks
benchmarks of several script languages found github

### Method
The tests were made on Win10 64 bits Desktop with Intel i7-8700 CPU @ 3.2 GHz and 16GB of RAM.
Linux was installed under Windows from [Win Store ](https://www.microsoft.com/en-us/p/ubuntu-1804-lts/9n9tngvndl3q?activetab=pivot:overviewtab "Win Store ")
- Description:    Ubuntu 18.04.2 LTS
- Release:        18.04
- Codename:       bionic

All the projects were compiled with:
- Linux gcc v7.4 when or Clang-3.9 when LLVM was required. -O3 was used in all projects.
- Win: Visual Studio 2019, target x64, optimization /O2
